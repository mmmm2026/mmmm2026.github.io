# Multilingual Minds and Machines 2026

Welcome to the official website for the **Multilingual Minds and Machines 2026** workshop.  
This site hosts key information including the call for abstracts, schedule, and contact details.

🌐 [Visit the live site](https://mmmm2026.github.io/)

## About

Multilingual Minds and Machines is a workshop exploring multilingual language processing in humans and machines, bringing together researchers in computational linguistics, psycholinguistics, and cognitive science.
